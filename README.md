宝贝成长记 - 婴儿成长记录软件
一款美观、易用的婴儿成长记录软件，帮助家长记录宝宝成长过程中的每一个珍贵瞬间。

功能特点
宝贝信息管理：记录宝贝的基本信息，包括姓名、出生日期、性别和头像
成长照片库：上传、管理和查看宝贝的成长照片，支持批量上传
成长瞬间记录：记录宝贝的第一次翻身、第一次说话等重要里程碑
成长日记：写下宝贝成长过程中的点滴故事和感受
本地数据存储：使用浏览器本地存储，数据安全不丢失
响应式设计：适配各种屏幕尺寸，手机和电脑都可以使用
美观界面：精心设计的界面，给您愉悦的使用体验
如何使用
在浏览器中打开 index.html 文件
首先点击「编辑信息」按钮，填写宝贝的基本信息
使用顶部或各功能区域的「添加」按钮开始记录
可以随时编辑或删除已添加的记录
照片可以点击查看大图
文件结构
index.html：应用的主要HTML结构
style.css：自定义样式表
script.js：交互功能实现
技术栈
HTML5
CSS3
JavaScript (原生)
Tailwind CSS
Font Awesome
注意事项
数据存储在浏览器的本地存储中，请不要清除浏览器数据
建议定期备份重要照片到其他存储设备
对于大量照片的上传，可能会受到浏览器内存限制
